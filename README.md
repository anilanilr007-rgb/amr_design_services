# AMR DESIGN SERVICE

Professional CAD & Engineering Solutions Website

---

## 📌 About

AMR DESIGN SERVICE is a professional engineering and CAD design service website developed using HTML, CSS, and JavaScript.

The website provides industrial design solutions including:
- 3D CAD Design
- 2D CAD Drafting
- Sheet Metal Design
- CAD Conversion
- Reverse Engineering
- Conceptual Design

Users can:
- Explore engineering services
- Submit project inquiries
- Upload CAD files and drawings
- Send technical requirements online
